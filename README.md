KFCZONE es una plataforma web estática que organiza y presenta contenido relacionado con videojuegos. El sistema no vende productos reales, sino que simula un entorno de catálogo, donde el usuario puede visualizar juegos, explorar categorías y utilizar un carrito de compras de carácter demostrativo.
El sistema está compuesto por seis secciones principales:
* Inicio de Sesión
* Tienda
* Ofertas
* Explorar
* Página individual de juego
* Carrito de Compras
Además, se incorpora un conjunto de estilos y scripts individuales y globales que controlan la navegación, el menú responsable y elementos compartidos en todas las páginas.
KFCZONE permite consultar imágenes, nombres, precios simulados y descripciones básicas de videojuegos mediante APIs, así como realizar acciones de agregar o eliminar elementos del carrito sin necesidad de backend.
